# Dados de entrada

Este diretório recebe a planilha de respostas mais recente exportada da coleta de campo.

- `Respostas - Pesquisa Economia Solidária - *.xlsx` — **não versionado no git** (contém PII:
  telefone, e-mail, CNPJ do empreendimento e nome/telefone da pessoa entrevistada). Guarde a
  cópia mestra em local de acesso controlado (ex.: drive institucional do NUPESER) e copie aqui
  apenas para rodar o pipeline localmente.
- `Necessidades - Dados Ecosol AVI.pdf` — ficha de necessidades dos stakeholders, não é sensível,
  pode ficar versionado.

Para atualizar os dados do dashboard com uma nova exportação, veja
`docs/como_atualizar_os_dados.md` na raiz do projeto.
